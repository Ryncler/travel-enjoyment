# TravelEnjoyment

毕设项目：“基于ABP和Vue框架的旅享服务平台的设计与实现”

采用ABP框架作为后端微服务，提供Web服务。

采用Vue框架作为前端服务，设计前台与后台，提供给用户使用。

数据库使用PostgreSQL，使用Minio作为S3对象存储服务。
